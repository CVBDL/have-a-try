Application introduction

About FTVP Device Management tools 

  This is a platform which could manage FTVP hardware devices, FTVP manager could add new devices or deleted unnecessary hardwares in this web site.
  Also the manager could assign tasks to testers and tracking the process in task tab.

User Security 

  User security function is not finished yet, and the hardware information is saved on local. Those features will be on going.

Common Questions 

  1. How to sign in?
  [Admin] Currently,the security function is not ready yet, everyone has admin authority. Security is on going.
  2. How to add new user?
  [Admin] Administrator has right to add new user and after the user added, he/she's name will be shown in name list when creating new devices.
